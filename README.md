# Calculadora de Média em Java

Programa desenvolvido para o curso de programação em Java que calcula a média de 4 notas e determina a situação do aluno.

## Funcionalidades
- Validação de notas (0 a 10)
- Cálculo de média aritmética
- Classificação: Aprovado/Recuperação/Reprovado
- Análise detalhada do desempenho

## Tecnologias utilizadas
- Java
- Estruturas de controle: if/else, while, for, switch
- Scanner para entrada de dados
